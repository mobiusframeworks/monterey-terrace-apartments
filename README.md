# Property Rental Website

A professional, responsive website template for rental properties. Cloned from Uptown Place Apartments design.

## Quick Start

1. **Add Your Images**
   - Place your property photos in the `/images` folder
   - See `/images/README.md` for required image names and sizes

2. **Customize the Content**
   - Open `index.html` in a text editor
   - Update the property name, address, and description
   - Modify contact information (phone, email)
   - Adjust vacancy listings with your actual units

3. **View the Website**
   - Open `index.html` in your web browser
   - Or use a local server for best results

## Files Included

- `index.html` - Main website structure
- `styles.css` - All styling and design
- `script.js` - Interactive features (mobile menu, smooth scrolling)
- `images/` - Folder for property photos

## Customization Guide

### Update Property Information

Find and replace these in `index.html`:

- **Property Name**: "Uptown Place Apartments"
- **Address**: "Big W 704 St Anns Outlake Street Chico, CA 95928"
- **Phone**: "(530) 123-4567"
- **Email**: "info@uptownplace.com"
- **Welcome Text**: Update the description in the Welcome Section

### Update Colors

The main colors are defined in `styles.css`:

- **Primary Orange**: `#ff6b35` (buttons, accents)
- **Dark Gray**: `#2c2c2c` (header, footer)
- **Light Background**: `#f9f9f9`

Search and replace these hex codes to change the color scheme.

### Add More Vacancy Listings

Copy the `.vacancy-card` div in the HTML and modify:
- Image source
- Bedroom/bathroom count
- Square footage
- Monthly price
