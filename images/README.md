# Images Folder

This folder should contain the property images for your website.

## Required Images:

1. **hero-apartments.jpg** - Main hero image showing the exterior of your property
   - Recommended size: 1920x500px or larger
   - This appears at the top of the homepage

2. **apartment-1.jpg** - First vacancy listing image
   - Recommended size: 800x600px or larger
   - Shows an available unit

3. **apartment-2.jpg** - Second vacancy listing image
   - Recommended size: 800x600px or larger
   - Shows another available unit

## Tips for Best Results:

- Use high-quality JPG images
- Keep file sizes optimized (under 500KB per image)
- Make sure images are well-lit and professionally photographed
- Use landscape orientation for hero image
- Vacancy images can be of building exterior, interior, or amenities

## How to Add Images:

1. Take photos of your property or hire a photographer
2. Edit/resize images as needed
3. Name them according to the list above
4. Place them in this folder
5. The website will automatically display them
